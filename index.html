    Two ways to measure pace — pick which one drives the projections

    Last 30 days is a rolling window — it always measures "today minus 30 days" through today, so it shifts forward every time you open the dashboard and reflects recent performance only. Full history is a fixed window — it averages across every uploaded report from the first to the most recent, so one slow or fast stretch gets blended in with everything else. Both are shown below for every rate; use this to choose which one the backlog projections (days-to-clear, target-date pace) are based on.
